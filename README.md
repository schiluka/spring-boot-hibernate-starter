Spring Boot starter
==================

A simple command line program with Hibernate/MySQL integration. It will insert/delete rows as passed in main program. 

Following are used

- JDK8
- Spring Boot 1.5.11
- Hibernate
- MySql

### Usage

mvn clean

mvn compile

mvn install -DskipTests

java -jar `<jar file name> <number of rows>`
